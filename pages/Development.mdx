## Local

In your `.mdx` files, import from `ui`:

```jsx
import { Button, LucideIcon } from "ui";

<Button leftIcon={<LucideIcon name="wallet" />}>Lorem Button</Button>;
```

## Dist build

Generally you don't have to do this.

In your `.mdx` files, import like this:

```jsx
import { Button, LucideIcon } from "../dist/nomimono";

<Button leftIcon={<LucideIcon name="wallet" />}>Lorem Button</Button>;
```

And import css in `_app.tsx` in `pages` folder:

```jsx
import "../dist/style.css";
import { AppProps } from "next/app";

function MyApp({ Component, pageProps }: AppProps) {
  return <Component {...pageProps} />;
}

export default MyApp;
```
