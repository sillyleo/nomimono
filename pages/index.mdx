# Welcome to Nomimono

A simple component library

## Usage

`pnpm install @sayaww/nomimono @fontsource/inter-tight`

```jsx
import { Button } from "@sayaww/nomimono";
import "@sayaww/nomimono/style.css";

<Button>Hello</Button>;
```

## Nomimono style wrapper

Use `.nomimono-style` for text styles other than builtin `Text` components.
